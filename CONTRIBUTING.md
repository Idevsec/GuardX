## **Contributing to GuardX**

We value your contributions! Whether it's fixing a bug, suggesting an enhancement, or adding a feature, we welcome all forms of participation to improve **GuardX**.

### **Getting Started**
1. **Fork the Repository**:
   - Visit the [GuardX GitHub repository](https://github.com/idevsec/GuardX).
   - Click the "Fork" button to create a copy of the repository in your GitHub account.

2. **Clone Your Fork**:
   ```bash
   git clone https://github.com/idevsec/GuardX.git
   cd GuardX
   ```

3. **Set Up Your Environment**:
   - Install the required dependencies:
     ```bash
     pip install -r requirements.txt
     ```

4. **Create a Branch**:
   - Use meaningful names for your branches:
     ```bash
     git checkout -b feature/add-new-functionality
     ```

---

### **Contributing Guidelines**
1. **Code Style**:
   - Adhere to [PEP 8](https://peps.python.org/pep-0008/) for Python code.
   - Use meaningful variable and function names.

2. **Document Changes**:
   - Update the `README.md` or add comments to explain new features.
   - If applicable, include examples or screenshots in your pull request.

3. **Testing**:
   - Test your changes thoroughly on supported platforms (Linux and Windows).
   - If possible, add automated tests for your code.

4. **Commit Standards**:
   - Write clear and concise commit messages:
     ```bash
     git commit -m "Add functionality to list active connections"
     ```

5. **Pull Requests**:
   - Push your branch to your fork:
     ```bash
     git push origin feature/add-new-functionality
     ```
   - Open a pull request from your branch to the `main` branch of the original repository.
   - Provide a detailed description of your changes.

### **Contribution Ideas**

- Enhance the GUI with advanced functionality (e.g., dark mode toggle, dashboard summaries).
- Implement real-time network monitoring.
- Integrate advanced security features like port scanning or intrusion detection.
- Translate the interface for multi-language support.
